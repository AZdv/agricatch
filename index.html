Agricatch

Data aggregation library for Django

Download .zip Download .tar.gz View on GitHub

This library started as an algorithm I wrote in Vanilla PHP, then moved to Yii, and finally came to my senses and rewrote the entire thing on Python (with a little help from my friend Django).

To get some sense into what's going on, try the following:

python manage.py syncdb
python manage.py doimport gizmodo
